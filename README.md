# RestTest Bench API Application

## Description

A simple webpage to display data fetched from Bench API

## To Start

```
npm install
npm run build
```

Open `dist/index.html` in your browser

## To Develop

```
npm run dev
```

Go to: `localhost:1234` in your browser
